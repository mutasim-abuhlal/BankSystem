# Bank Management System

## Overview

Bank Management System is a console-based application developed in C++ using a functional programming approach. The system provides basic banking operations such as account management, deposits, withdrawals, money transfers, and customer information management.

The project focuses on modular design by organizing functionality into separate functions rather than relying on object-oriented programming concepts.

---

## Features

* Create new customer accounts
* Deposit funds
* Withdraw funds
* Transfer money between accounts
* Search for accounts
* View customer information
* Update account details
* Delete accounts
* Display all accounts
* Store and retrieve data using file handling

## How to Build and Run

### Requirements

* C++ Compiler (GCC, MinGW, MSVC, or Clang)
* C++11 or later

### Compile

```bash
g++ main.cpp -o BankSystem
```

### Run

Linux/macOS:

```bash
./BankSystem
```

Windows:

```bash
BankSystem.exe
```
